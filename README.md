# RNe2eTestCI

## install npm packages

yarn

## run js server

yarn start

## run android / ios

yarn android
yarn ios

## run e2e detox on iOS simulator

yarn run e2e:ios:build
yarn run e2e:ios:run

## run e2e test during dev (use running app instead of installing app)

yarn run e2e:ios:run-reuse
